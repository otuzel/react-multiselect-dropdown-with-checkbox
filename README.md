React component assignment: Multiselect dropdown with checkbox

## Installation

In the project directory, run:

### `npm install`

## Usage

In the project directory, run:

### `npm start`

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
